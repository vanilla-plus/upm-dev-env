# Vanilla First Person Kit

Vanilla First Person Kit is part of the Vanilla For Unity SDK.

Vanilla First Person Kit is a series of tools for creating first-person experiences.

---

## Installation

Vanilla Plus packages are installed through Unity's Package Manager using a [scoped registry](https://docs.unity3d.com/Manual/upm-scoped.html). Open your Unity Project of choice and select:

> Edit menu > Project settings > Package Manager > Scoped Registries > Plus button

Then add:


	name:      Vanilla Plus
	url:       http://35.231.76.113:4873/
	Scopes:    vanilla.plus

---

## Debugging

You can enable debugging for this package by including the keywords VANILLA and FIRST_PERSON_KIT in the Unity Editor scripting define symbols.
